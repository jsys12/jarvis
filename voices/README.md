# Клонирование голоса (XTTS-v2)

Положите сюда файл **jarvis.wav** — и Феникс заговорит этим голосом.

Требования к референсу:
- 10–30 секунд **чистой речи** одного человека: без музыки, эффектов и шумов;
- WAV (моно/стерео, любой sample rate — конвертируется само);
- чем спокойнее и ровнее речь в сэмпле, тем стабильнее клон.

Для «голоса Джарвиса из фильма» вырежьте фрагменты реплик Джарвиса из
русского дубляжа (например, через Audacity), склейте куски без фоновой
музыки в один файл и сохраните как `voices/jarvis.wav`.

Управление в config.json:
- `"tts_backend": "auto"` — клон включится сам, как только файл появится;
- `"tts_backend": "piper"` — вернуться на быстрый нейроголос (~0.2 с против
  ~2-4 с у клона);
- `"xtts_ref"` — другой путь к референсу.

Клонирование работает локально на GPU (XTTS-v2, ~2 ГБ VRAM). Только для
личного использования: голоса актёров защищены правами.
